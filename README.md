# My-C-Language-Mastery
A transparent log of my journey to master the C programming language - from fundamentals to advanced topics. (这是我的C语言精通之路的透明记录——从基础到高级主题。)
> **A Transparent Journey from Fundamentals to Proficiency**
> 本仓库是我系统学习C语言的完整记录，旨在通过**透明、持续、可验证**的方式，彻底掌握C语言的核心精髓。

## 当前学习重心
1.  **指针与内存管理** - 理解地址、指针运算、动态内存分配。
2.  **数据结构** - 亲手实现链表、栈、队列、树。
3.  **文件I/O与标准库** - 掌握文件操作和常用标准库函数。
4.  **算法入门** - 将常见算法用C语言实现。
